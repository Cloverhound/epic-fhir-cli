The `familyhistory` command manages FamilyMemberHistory resources — family health history. This page also covers related person, consent, provenance, and questionnaire response commands.

## FamilyMemberHistory

**Aliases:** `family-history`, `fmh`

### `familyhistory get <id>`

Fetch a family member history record by ID.

```bash
fhir-cli familyhistory get abc123
```

### `familyhistory search`

Search for family member history records.

```bash
fhir-cli familyhistory search --patient e63wRTbPfr1p8UW81d8Seiw3
fhir-cli familyhistory search --patient e63wRTbPfr1p8UW81d8Seiw3 --status completed
fhir-cli familyhistory search --patient e63wRTbPfr1p8UW81d8Seiw3 --relationship parent
```

| Flag | Description |
|------|-------------|
| `--patient` | Patient FHIR ID |
| `--status` | Status: `completed`, `partial` |
| `--relationship` | Relationship to patient |
| `--code` | Condition code |
| `--count` | Maximum number of results (default: `20`) |

### `familyhistory list <patient-id>`

List family member history for a patient.

```bash
fhir-cli familyhistory list e63wRTbPfr1p8UW81d8Seiw3
```

| Flag | Description |
|------|-------------|
| `--count` | Maximum number of results (default: `50`) |

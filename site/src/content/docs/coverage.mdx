The `coverage` command manages Coverage resources — insurance and payment information. This page also covers ExplanationOfBenefit and ServiceRequest commands.

## Coverage

**Aliases:** `insurance`

### `coverage get <id>`

Fetch a coverage record by ID.

```bash
fhir-cli coverage get abc123
```

### `coverage search`

Search for coverage records.

```bash
fhir-cli coverage search --patient e63wRTbPfr1p8UW81d8Seiw3
fhir-cli coverage search --patient e63wRTbPfr1p8UW81d8Seiw3 --status active
```

| Flag | Description |
|------|-------------|
| `--patient` | Patient FHIR ID |
| `--beneficiary` | Beneficiary FHIR ID |
| `--status` | Status: `active`, `cancelled` |
| `--type` | Coverage type |
| `--payor` | Payor organization |
| `--count` | Maximum number of results (default: `20`) |

### `coverage list <patient-id>`

List coverage records for a patient.

```bash
fhir-cli coverage list e63wRTbPfr1p8UW81d8Seiw3
```

| Flag | Description |
|------|-------------|
| `--count` | Maximum number of results (default: `20`) |

## Quick Install

### macOS / Linux

```bash
curl -fsSL https://raw.githubusercontent.com/CloverhoundPS/epic-fhir-cli/main/install.sh | sh
```

This downloads the latest release for your platform, extracts it to `~/.local/bin`, and adds it to your PATH if needed.

### Windows (PowerShell)

```powershell
irm https://raw.githubusercontent.com/CloverhoundPS/epic-fhir-cli/main/install.ps1 | iex
```

Installs to `%LOCALAPPDATA%\fhir-cli` and adds it to your user PATH.

## Manual Download

Download the appropriate archive from the [GitHub Releases](https://github.com/CloverhoundPS/epic-fhir-cli/releases) page.

| Platform      | Architecture | File                                    |
|---------------|--------------|-----------------------------------------|
| macOS         | Apple Silicon | `fhir-cli_VERSION_darwin_arm64.tar.gz` |
| macOS         | Intel        | `fhir-cli_VERSION_darwin_amd64.tar.gz` |
| Linux         | x86_64       | `fhir-cli_VERSION_linux_amd64.tar.gz`  |
| Linux         | ARM64        | `fhir-cli_VERSION_linux_arm64.tar.gz`  |
| Windows       | x86_64       | `fhir-cli_VERSION_windows_amd64.zip`   |
| Windows       | ARM64        | `fhir-cli_VERSION_windows_arm64.zip`   |

Extract the archive and place the `fhir-cli` binary somewhere on your PATH.

## Build from Source

Requires [Go 1.21+](https://go.dev/dl/).

```bash
git clone https://github.com/CloverhoundPS/epic-fhir-cli.git
cd epic-fhir-cli
make build
```

The binary is placed in `./bin/fhir-cli`. To install it to your `$GOPATH/bin`:

```bash
make install
```

## Verify Installation

```bash
fhir-cli version
```

The `medication` command manages MedicationRequest resources â€” prescriptions and medication orders. It also provides subcommands for MedicationStatement, MedicationAdministration, and MedicationDispense resources.

## Commands

### `medication get <id>`

Fetch a single medication request by ID.

```bash
fhir-cli medication get abc123
```

### `medication search`

Search for medication requests.

```bash
fhir-cli medication search --patient e63wRTbPfr1p8UW81d8Seiw3
fhir-cli medication search --patient e63wRTbPfr1p8UW81d8Seiw3 --status active
fhir-cli medication search --patient e63wRTbPfr1p8UW81d8Seiw3 --intent order
```

| Flag | Description |
|------|-------------|
| `--patient` | Patient FHIR ID |
| `--status` | Status: `active`, `on-hold`, `cancelled`, `completed`, `stopped` |
| `--intent` | Intent: `order`, `proposal`, `plan` |
| `--authoredon` | Authored date range |
| `--count` | Maximum number of results (default: `20`) |

### `medication active <patient-id>`

Shortcut to get active medications (pre-sets `status=active`).

```bash
fhir-cli medication active e63wRTbPfr1p8UW81d8Seiw3
```

| Flag | Description |
|------|-------------|
| `--count` | Maximum number of results (default: `50`) |

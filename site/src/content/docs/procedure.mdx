---
title: Procedure
description: Query actions performed on or for a patient.
---

The `procedure` command manages Procedure resources — actions performed on or for a patient.

## Commands

### `procedure get <id>`

Fetch a single procedure by ID.

```bash
fhir-cli procedure get abc123
```

### `procedure search`

Search for procedures.

```bash
fhir-cli procedure search --patient e63wRTbPfr1p8UW81d8Seiw3
fhir-cli procedure search --patient e63wRTbPfr1p8UW81d8Seiw3 --status completed
fhir-cli procedure search --patient e63wRTbPfr1p8UW81d8Seiw3 --code 80146002
fhir-cli procedure search --patient e63wRTbPfr1p8UW81d8Seiw3 --date ge2024-01-01
```

| Flag | Description |
|------|-------------|
| `--patient` | Patient FHIR ID |
| `--status` | Status: `completed`, `in-progress`, `not-done` |
| `--code` | Procedure code (CPT, SNOMED) |
| `--date` | Date range |
| `--category` | Category |
| `--count` | Maximum number of results (default: `20`) |

### `procedure list <patient-id>`

List procedures for a patient.

```bash
fhir-cli procedure list e63wRTbPfr1p8UW81d8Seiw3
```

| Flag | Description |
|------|-------------|
| `--count` | Maximum number of results (default: `20`) |
